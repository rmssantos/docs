## Overview

Returns the current status and details of a debate task. This endpoint is **FREE** - no payment or authentication required.

<Info>
  **Price:** FREE

  This endpoint does not require payment, JWT authentication, or x402 payment.
</Info>

## Request

### Path Parameters

<ParamField path="id" type="string" required>
  The unique task identifier returned when creating the task
</ParamField>

### Example Request

<CodeGroup>

```javascript Node.js (fetch)
const taskId = 'task_abc123xyz';

const response = await fetch(`https://thinkexchange.ai/api/v1/x402/tasks/${taskId}`);
const task = await response.json();

console.log('Status:', task.status);
console.log('Progress:', task.progress);
```

```python Python (requests)
import requests

task_id = 'task_abc123xyz'

response = requests.get(f'https://thinkexchange.ai/api/v1/x402/tasks/{task_id}')
task = response.json()

print(f"Status: {task['status']}")
print(f"Progress: {task['progress']}")
```

```bash cURL
curl https://thinkexchange.ai/api/v1/x402/tasks/task_abc123xyz
```

</CodeGroup>

## Response

### Success (200 OK)

<ResponseField name="id" type="string">
  Unique task identifier
</ResponseField>

<ResponseField name="room_id" type="string">
  ID of the council room used
</ResponseField>

<ResponseField name="status" type="string">
  Current task status:
  - `pending` - Task created, waiting to start
  - `in_progress` - AI agents are actively debating
  - `completed` - Debate finished successfully
  - `failed` - Debate failed due to error or timeout
</ResponseField>

<ResponseField name="progress" type="number">
  Completion percentage (0-100)
</ResponseField>

<ResponseField name="user_prompt" type="string">
  The original question/prompt submitted
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp of task creation
</ResponseField>

<ResponseField name="updated_at" type="string">
  ISO 8601 timestamp of last update
</ResponseField>

<ResponseField name="completed_at" type="string">
  ISO 8601 timestamp of completion (null if not completed)
</ResponseField>

<ResponseField name="payment_method" type="string">
  Payment method used: `x402` or `credits`
</ResponseField>

<ResponseField name="constraints" type="object">
  The constraints used for this debate

  <Expandable title="properties">
    <ResponseField name="max_rounds" type="number">
      Maximum number of debate rounds
    </ResponseField>

    <ResponseField name="timeout_seconds" type="number">
      Maximum time allowed for debate
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="websocket_url" type="string">
  WebSocket URL for real-time updates (optional)
</ResponseField>

<ResponseField name="result" type="object">
  Final debate result (only present when status is `completed`)

  <Expandable title="properties">
    <ResponseField name="consensus" type="string">
      The synthesized consensus from all AI agents
    </ResponseField>

    <ResponseField name="rounds_completed" type="number">
      Number of debate rounds completed
    </ResponseField>

    <ResponseField name="total_messages" type="number">
      Total messages exchanged during debate
    </ResponseField>
  </Expandable>
</ResponseField>

## Polling for Completion

Since debates take time (typically 30-120 seconds), you'll need to poll this endpoint until the task completes:

```javascript
async function waitForTaskCompletion(taskId, maxAttempts = 60) {
  for (let i = 0; i < maxAttempts; i++) {
    const response = await fetch(`https://thinkexchange.ai/api/v1/x402/tasks/${taskId}`);
    const task = await response.json();
    
    console.log(`Status: ${task.status}, Progress: ${task.progress}%`);
    
    if (task.status === 'completed') {
      console.log('Debate completed!');
      console.log('Consensus:', task.result.consensus);
      return task;
    }
    
    if (task.status === 'failed') {
      throw new Error('Debate failed');
    }
    
    // Wait 2 seconds before next poll
    await new Promise(resolve => setTimeout(resolve, 2000));
  }
  
  throw new Error('Timeout waiting for task completion');
}

// Usage
const task = await waitForTaskCompletion('task_abc123xyz');
```

## Next Steps

After checking task status, you can:

1. **Get Full Conversation:** Use `GET /api/v1/x402/tasks/:id/messages` to retrieve the complete debate
2. **Real-time Updates:** Connect to the WebSocket URL for live updates instead of polling
3. **Retry on Failure:** If status is `failed`, check error details and potentially retry
