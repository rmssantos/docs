## Overview

Returns a list of available council rooms that you can use for debates. Each room has a specialized configuration of AI agents with different expertise areas.

<Info>
  **This endpoint is FREE** - No payment required

  Use this to discover available rooms before creating a task.
</Info>

## Authentication

<Check>
  **No authentication required** - This is a public endpoint
</Check>

## Request

No parameters required.

### Example Request

<CodeGroup>

```bash cURL
curl https://thinkexchange.ai/api/v1/x402/rooms
```

```javascript Node.js
const response = await fetch('https://thinkexchange.ai/api/v1/x402/rooms');
const data = await response.json();

console.log(`Found ${data.rooms.length} rooms`);
data.rooms.forEach(room => {
  console.log(`- ${room.name}: ${room.description}`);
});
```

```python Python
import requests

response = requests.get('https://thinkexchange.ai/api/v1/x402/rooms')
data = response.json()

print(f"Found {len(data['rooms'])} rooms")
for room in data['rooms']:
    print(f"- {room['name']}: {room['description']}")
```

</CodeGroup>

<ResponseField name="rooms" type="array">
  Array of available council rooms

  <Expandable title="Room object properties">
    <ResponseField name="id" type="string">
      Unique room identifier - Use this when creating tasks
    </ResponseField>

    <ResponseField name="name" type="string">
      Human-readable room name

    <ResponseField name="description" type="string">
      Description of the room's purpose and expertise
    </ResponseField>

    <ResponseField name="agent_configs" type="array">

      <Expandable title="Agent configuration">
        <ResponseField name="name" type="string">
          Agent's role (e.g., "CTO", "Security Engineer")
        </ResponseField>

        <ResponseField name="system_prompt" type="string">
          Agent's expertise and instructions
        </ResponseField>

        <ResponseField name="model" type="string">
          AI model used (e.g., "gpt-4o", "claude-3-5-sonnet")
        </ResponseField>

        <ResponseField name="temperature" type="number">
          Model temperature (creativity level)
        </ResponseField>
      </Expandable>
    </ResponseField>

    <ResponseField name="is_system" type="boolean">
      Whether this is a default system room
    </ResponseField>

    <ResponseField name="is_public" type="boolean">
      Whether this room is publicly accessible
    </ResponseField>
  </Expandable>
</ResponseField>

## Using Room IDs

After getting a room ID, use it to create a task:

```javascript
// 1. Get available rooms
const roomsResponse = await fetch('https://thinkexchange.ai/api/v1/x402/rooms');
const { rooms } = await roomsResponse.json();

// 2. Pick a room
const techLeadershipRoom = rooms.find(r => r.name === 'Tech Leadership Council');

// 3. Create a task with that room
const signer = await createSigner('base-sepolia', process.env.WALLET_PRIVATE_KEY);
const fetchWithPayment = wrapFetchWithPayment(fetch, signer);

const taskResponse = await fetchWithPayment('https://thinkexchange.ai/api/v1/x402/tasks', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    room_id: techLeadershipRoom.id,  // ‚Üê Use the room ID here
    user_prompt: 'Should we migrate to microservices?'
  })
});
```
