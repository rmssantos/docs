## Overview

Returns a list of all available specialized council rooms. This endpoint is **FREE** - no payment or authentication required.

<Info>
  **Price:** FREE

  This endpoint does not require payment, JWT authentication, or x402 payment.
</Info>

## Request

No parameters required. Simply make a GET request to the endpoint.

### Example Request

<CodeGroup>

```javascript Node.js (fetch)
const response = await fetch('https://thinkexchange.ai/api/v1/x402/rooms');
const data = await response.json();

console.log(`Found ${data.rooms.length} available rooms`);
data.rooms.forEach(room => {
  console.log(`- ${room.name}: ${room.description}`);
});
```

```python Python (requests)
import requests

response = requests.get('https://thinkexchange.ai/api/v1/x402/rooms')
data = response.json()

print(f"Found {len(data['rooms'])} available rooms")
for room in data['rooms']:
    print(f"- {room['name']}: {room['description']}")
```

```bash cURL
curl https://thinkexchange.ai/api/v1/x402/rooms
```

</CodeGroup>

## Usage Example

Typically, you'll use this endpoint to:
1. Discover available rooms before creating a debate
2. Let users select a specialized council
3. Cache room information in your application

```javascript
// Complete workflow example
async function createDebateWithRoomSelection(userPrompt) {
  // 1. Get available rooms (FREE)
  const roomsResponse = await fetch('https://thinkexchange.ai/api/v1/x402/rooms');
  const { rooms } = await roomsResponse.json();
  
  // 2. Filter for active crypto-related room
  const cryptoRoom = rooms.find(r => r.specialty === 'crypto' && r.is_active);
  
  if (!cryptoRoom) {
    throw new Error('No crypto room available');
  }
  
  // 3. Create debate task with selected room (PAID - $0.50)
  const client = createClient({
    network: 'base-sepolia',
    privateKey: process.env.WALLET_PRIVATE_KEY,
  });
  
  const taskResponse = await client.fetch('https://thinkexchange.ai/api/v1/x402/tasks', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      room_id: cryptoRoom.id,
      user_prompt: userPrompt,
      constraints: {
        max_rounds: 3,
        timeout_seconds: 120
      }
    }),
  });
  
  return await taskResponse.json();
}
```

## Next Steps

After getting the list of rooms, you can:

1. **Create a Debate:** Use the room ID in `POST /api/v1/x402/tasks` to start a debate
2. **Filter Rooms:** Select rooms based on specialty or name
3. **Cache Results:** Rooms rarely change, safe to cache for a few hours
