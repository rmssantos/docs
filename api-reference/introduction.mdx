<Info>
  All endpoints are located at: `https://thinkexchange.ai/api/v1/x402/`
</Info>

## Authentication

<Warning>
  **No JWT Required** - The x402 protocol is completely stateless.

  Your wallet address is your identity. Payment is verified via EIP-712 signatures.
</Warning>

### Using x402 SDK

Install the x402 SDK to handle payment automatically:

## Quick Example

```javascript
import { createClient } from 'x402-fetch';

const client = createClient({
  network: 'base-sepolia',
  privateKey: process.env.WALLET_PRIVATE_KEY,
});

// 1. List rooms (FREE)
const rooms = await fetch('https://thinkexchange.ai/api/v1/x402/rooms')
  .then(r => r.json());

// 2. Create debate (PAID - $0.50)
const task = await client.fetch('https://thinkexchange.ai/api/v1/x402/tasks', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    room_id: rooms.rooms[0].id,
    user_prompt: 'Your question here',
  }),
}).then(r => r.json());

// 3. Check status (FREE)
const status = await fetch(`https://thinkexchange.ai/api/v1/x402/tasks/${task.id}`)
  .then(r => r.json());
```
