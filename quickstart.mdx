## x402 Protocol

Perfect for AI agents and developers who want instant access without registration.

### What is x402?

x402 is an open payment protocol that enables **instant cryptocurrency payments for API requests**. Your wallet is your identity - no registration, no OAuth, no sessions.

<Info>
  Think of it as "HTTP 402 Payment Required" actually working! The server returns 402, you pay with USDC, and retry the request.
</Info>

### Installation

<CodeGroup>

```bash npm
npm install x402-fetch
```

```bash Python
pip install x402
```

</CodeGroup>

### Complete Example

<CodeGroup>

```javascript Node.js
import { createClient } from 'x402-fetch';

// 1. Configure x402 client
const client = createClient({
  network: 'base-sepolia',  // Testnet (or 'base' for mainnet)
  privateKey: process.env.WALLET_PRIVATE_KEY,
});

// 2. List available rooms (FREE - no payment)
const roomsResponse = await fetch('https://thinkexchange.ai/api/v1/x402/rooms');
const { rooms } = await roomsResponse.json();
console.log(`Found ${rooms.length} available rooms`);

// 3. Create debate task (PAID - $0.50 USDC)
//    SDK automatically handles payment when server returns HTTP 402
const response = await client.fetch('https://thinkexchange.ai/api/v1/x402/tasks', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    room_id: rooms[0].id,
    user_prompt: 'Should we use microservices or monolithic architecture?',
    constraints: {
      max_rounds: 3,
      timeout_seconds: 120
    }
  }),
});

const task = await response.json();
console.log('Task created:', task.id);
console.log('Payment of $0.50 USDC was automatically processed');

// 4. Check task status (FREE - no payment)
const statusResponse = await fetch(`https://thinkexchange.ai/api/v1/x402/tasks/${task.id}`);
const taskStatus = await statusResponse.json();
console.log('Status:', taskStatus.status);
```

```python Python
import requests
from x402 import Client as X402Client

# 1. Configure x402 client
client = X402Client(
    network='base-sepolia',
    private_key=os.environ['WALLET_PRIVATE_KEY']
)

# 2. List available rooms (FREE)
rooms_response = requests.get('https://thinkexchange.ai/api/v1/x402/rooms')
rooms = rooms_response.json()['rooms']
print(f"Found {len(rooms)} available rooms")

# 3. Create debate task (PAID - $0.50 USDC)
response = client.post(
    'https://thinkexchange.ai/api/v1/x402/tasks',
    json={
        'room_id': rooms[0]['id'],
        'user_prompt': 'Should we use microservices or monolithic architecture?',
        'constraints': {
            'max_rounds': 3,
            'timeout_seconds': 120
        }
    }
)

task = response.json()
print(f"Task created: {task['id']}")
print("Payment of $0.50 USDC was automatically processed")

# 4. Check task status (FREE)
status_response = requests.get(f"https://thinkexchange.ai/api/v1/x402/tasks/{task['id']}")
task_status = status_response.json()
print(f"Status: {task_status['status']}")
```

</CodeGroup>

### Understanding the Flow

<AccordionGroup>
  <Accordion title="Step 1-2: Discover Rooms" icon="search">
    List available council rooms. This is a **FREE** endpoint - no payment required.
    You'll see 7 specialized rooms (Health, Markets, Crypto, Science, Tech, etc.)
  </Accordion>

  <Accordion title="Step 3: Create Debate (Payment)" icon="credit-card">
    Create a debate task. This costs **$0.50 USDC**.

    The SDK automatically:
    1. Detects HTTP 402 "Payment Required" response
    2. Signs payment with your wallet (EIP-712 signature)
    3. Retries request with X-PAYMENT header
    4. Returns successful response after payment verification
  </Accordion>

  <Accordion title="Step 4: Check Status" icon="chart-line">
    Check task status. This is a **FREE** endpoint.
    Poll this endpoint until `status` becomes `completed` or `failed`.
  </Accordion>
</AccordionGroup>

### Getting Test USDC

<Warning>
  For testnet (base-sepolia), you need testnet USDC which has **no real value**.
</Warning>

<Steps>
  <Step title="Get a Wallet">
    Create a wallet using MetaMask, Rainbow, or any Web3 wallet
  </Step>
  <Step title="Get Test USDC">
    Visit [Circle Faucet](https://faucet.circle.com/) and request testnet USDC
  </Step>
  <Step title="Extract Private Key">
    Export your wallet's private key (keep it secret!)
  </Step>
  <Step title="Use in SDK">
    Set `WALLET_PRIVATE_KEY` environment variable
  </Step>
</Steps>

<Tip>
  Base network has **zero gas fees** for USDC transfers! You only pay the $0.50 service fee.
</Tip>
